Question-1: What are the different ways to select an element in the DOM?
There are many ways to select an element in the DOM. These are-
  • 1. getElementById()
  • 2. getElementsByClassName
  • 3. getElementsByTagName
  • 4. querySelector
  • 5. querySelectorAll
Question-2: What is the difference between innerHTML, innerText, and textContent ?
1. innerHTML: We can get or set HTML element by innerHTML. It return the HTML code of a element.
2.innerText: We can get or set the visible content of a element by innerText. It ignores the hidden elements and scripts.
3.textContent: We can get or set the visible content of a element by innerText. It includes the hidden elements and scripts.
Question-3: What is event delegation in the DOM?
Event delegation is a powerful technique in JavaScript. Normally we attached multiple event listener to child node. But by event delegation we attached a single event listener to an element and access all the child node and parent node.
Question-4: What is event bubbling in the DOM?
Event bubbling is a type of DOM event propagation method. It starts with the target element and return back to it's parent element until it reach the root element
Question-5: How do you create, add, and remove elements using JavaScript?
Create: We can create an element by using document.createElement() method.
Add: We can add element by using appendChild(), innerHTML, innerText etc. method.
Remove: We can remove element by using remove(), removeChild() etc. method.
